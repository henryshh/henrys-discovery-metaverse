## 系统简介

### 概述

拧紧曲线智能分析系统是一个基于深度学习的工业质量分析工具，用于：
- 🔍 分析拧紧曲线数据
- 📊 识别不同的拧紧模式
- 🤖 预测拧紧质量（OK/NOK）
- 📈 可视化展示分析结果

### 核心功能

| 功能 | 说明 |
|------|------|
| **数据探索** | 浏览和筛选拧紧曲线数据 |
| **3D可视化** | 交互式3D图表展示 |
| **聚类分析** | 基于DTW算法识别5种拧紧模式 |
| **预测分析** | CNN模型预测拧紧质量 |
| **聚类代表对比** | 多维可视化对比各聚类特征 |

### 技术规格

- **数据格式**: JSON (DeMeter API标准格式)
- **数据点**: 500点/条曲线
- **通道**: 扭矩、角度、电流、标记
- **模型**: CNN (PyTorch)
- **聚类**: DTW + 层次聚类
- **准确率**: 100% (验证集)

## 安装指南

### 系统要求

- **操作系统**: Windows 10/11 64位
- **Python**: 3.11+ (需预先安装)
- **内存**: 4GB+
- **磁盘空间**: 2GB+
- **浏览器**: Chrome/Edge/Firefox 最新版

### 安装步骤

#### 方式一：离线安装包（推荐）

1. **获取安装包**
   - 文件: `tightening-ai-offline.zip`
   - 大小: 约 495 MB

2. **解压安装包**
   ```
   解压 tightening-ai-offline.zip 到任意目录
   例如: C:\TighteningAI\
   ```

3. **运行安装脚本**
   ```
   双击 install.bat
   等待安装完成（约5-10分钟）
   ```

4. **验证安装**
   ```
   安装完成后，命令行会显示 "Installation Complete!"
   ```

#### 方式二：源码安装

1. **克隆代码**
   ```bash
   git clone <repository-url>
   cd tightening-ai
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **验证数据**
   - 确保 `API/Anord.json` 存在
   - 确保 `output/cnn_model.pth` 存在

## 训练新数据

### 准备工作

1. **准备数据文件**
   - 格式: JSON
   - 标准: DeMeter API格式
   - 位置: 替换 `API/Anord.json`

2. **数据格式要求**
   ```json
   {"model": {
     "resultNumber": 1,
     "report": "OK",
     "results": [{
       "curves": [{
         "data": {
           "points": [
             {"torque": 0.1, "angle": 0, "current": 0.5, "marker": 0}
           ]
         }
       }]
     }]
   }}
   ```

### 训练步骤

#### 方式一：使用脚本

```bash
double-click train_new_data.bat
```

#### 方式二：命令行

```bash
# 1. 训练CNN模型
python train.py --model cnn --epochs 50 --save_model

# 2. 运行DTW聚类
python dtw_cluster_full.py

# 3. 重新启动应用
```

### 训练参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--model` | 模型类型: cnn, tcn, transformer | cnn |
| `--epochs` | 训练轮数 | 50 |
| `--batch_size` | 批次大小 | 16 |
| `--val_split` | 验证集比例 | 0.2 |
| `--save_model` | 保存模型 | - |

### 训练输出

- 模型文件: `output/cnn_model.pth`
- 聚类结果: `output/dtw_full_results.json`
- 训练历史: `output/cnn_history.json`

## 常见问题

### Q1: 安装时提示 pip 不是内部命令

**解决**: 确保 Python 和 pip 已正确安装并添加到环境变量

```bash
# 验证安装
python --version
pip --version
```

### Q2: 运行时提示缺少依赖

**解决**: 重新运行安装脚本

```bash
install.bat
```

### Q3: 浏览器无法打开 localhost:8501

**解决**:
1. 检查防火墙设置
2. 尝试手动访问 http://127.0.0.1:8501
3. 检查端口是否被占用

### Q4: 预测结果不准确

**可能原因**:
- 数据格式不正确
- 数据未归一化
- 模型需要重新训练

**解决**:
1. 检查JSON格式
2. 使用与训练数据相同的预处理
3. 用新数据重新训练模型

### Q5: 聚类结果与预期不符

**说明**: DTW聚类基于曲线形状相似度，不是简单的数值分类

**建议**: 查看聚类代表对比页面，理解各聚类的实际特征
